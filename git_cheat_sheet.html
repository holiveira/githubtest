Clone

Clonar repositório remoto
$ git clone http://user@github.com/path/to/repo

Clone Branch

Clonar branch de repositório remoto
$ git clone -b <branch> http://user@github.com/path/to/repo

Create

Criar novo repositório
$ git init

Commit

"Commitar" atualizações adcionando no índice
$ git commit -a -m "Message"

Push

Adicionar alterações ao repositório remoto
$ git push origin <branch>

Pull

Atualizar repositório local a partir do remoto
$ git pull

Add

Adicionar arquivo ao índice
$ git add <arquivo>

Add

Adicionar todos os arquivos .txt ao índice
$ git add '*.txt'

Branch

Identificar branch atual
$ git branch

Branch

Alterar para um branch específico
$ git checkout <branch>

Merge

Visualisar diferenças entre branchs
$ git diff <source_branch> <target_branch>

Merge

Merge de um outro branch
$ git merge <branch>

Status

Verificar status do índice
$ git status

Restore

Substituir arquivo pela última versão do HEAD
$ git checkout -- <arquivo>
